*************************** PROJECT MANAGE: DATABASE ******************************
#Cấu tạo:
	 $ user				: id(PK), username, pass, deleted
 	 $ profile			: date, email, address, github, user_id (FK)
	 $ project			: id(PK), leader_id, name, status, work_counts, deadline, introduction, deleted, created_date
	 $ user_project		: joined_date, leader, user_id (FK), project_id(FK)
	 $ work				: id(PK), name, status, introduction, deadline, deleted, created_date
	 $ comment			: int(PK), content, title, deleted, date, user_id (FK), work_id (FK), work_project_id(FK)
	 $ user_work		: work_code, user_id(FK), work_id(FK), work_project_id(FK)
	 $ leader_submit_work	: submited, title, content, deleted, user_id (FK), work_id(FK), work_project_id(20)

#Giải thích các mối liên kết
	$ Mỗi user đều có profiles riêng
	$ Một user sẽ thuộc hoặc không thuộc một project nào đó
	   Mỗi project cần phải có leader
	$ Mỗi user sẽ có hoặc không có công việc trong một project nào đó.
	   Mỗi work cần phải thuộc về một hoặc nhiều user
	$ Mỗi project có ít nhất là một work
	   Mỗi work cần thuộc về một project xác định
	$ Mỗi user sẽ có một hoặc nhiều cmt và một cmt chỉ thuộc về một người.
	$ Mỗi work sẽ hoặc có hoặc có rất nhiều cmt
